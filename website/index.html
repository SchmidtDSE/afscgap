Python Tools for AFSC GAP

Community contributed Python-based tools for working with public bottom trawl surveys data from the NOAA Alaska Fisheries Science Center Groundfish Assessment Program (NOAA AFSC GAP). These datasets from multiple locations provide information about where certain species were seen and when under what conditions, information useful for research in ocean health. In addition to visual analytics tools that do not require code, it provides Pythonic access to this information with automated pagination, convenient contextual documentation, memory efficient zero catch (absence data) inference, and easy query generation / emulation for data filtering.
